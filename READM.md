# Network Security Lab – Suricata & ELK

## 📌 Project Overview
Deployment of a monitored lab network using Suricata IDS and ELK Stack (Elasticsearch, Logstash, Kibana).

## 🎯 Objectives
- Monitor network traffic
- Detect suspicious activity using Suricata IDS
- Centralize logs with ELK
- Visualize alerts in Kibana dashboards

## 🛠 Technologies Used
- Suricata
- Elasticsearch
- Logstash
- Kibana
- Kali Linux
- VirtualBox
- TCP/IP Analysis

## 🔎 Features Implemented
- IDS deployment and configuration
- Traffic capture and analysis
- Log forwarding to ELK
- Dashboard creation
- Basic incident detection simulation
